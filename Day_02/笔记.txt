1.数组
	一维数组
	当你要定义多个变量的时候，就使用数组
	定义方法
	1.1完全赋值法
		int a[10] = {0,1,2,3,4,5,6,,,,}; int a， b， c， d。。。。 10个；
		int a[4] = {0,0,0,0};
		访问的时候数组的下标是从0开始
		a[5]的值就是5
	1.2不完全赋值法
		int a[10] = {0}; //这样的话 就是将第一个也就是a[0]赋值为0，后面的也自动全部赋值为0
	使用
	例子 printf("%d",a[3]);
2.练习
	从屏幕输入5个数给一个数组，然后把这个5个数按从大到小输出出来或者从小到大

4.二维数组
	他的本质也是一个一维数组

1.指针
	是C语言的灵魂
	6.1指针的定义方法
		int *p  //一个指针类型的变量
		指着是用来指向一个变量的地址的

3.函数
	函数的类型 函数名字（函数的参数（形参））
	函数的返回值
	
	子函数
		子函数之间可以相互调用
		子函数不可以调用主函数
		主函数可以随意调用子函数
	函数的传参
		非数组传参
		如果只要形参的数值的时候那么就只要传那个那个实参变量
		如果我们需要改变实参的值的时候，那么就要取那个实参的地址传进去
	
		数组传参
		数组名在作为参数传参的时候它会退化为指针
